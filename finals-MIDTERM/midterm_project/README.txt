Project Title: Dictionary Application
API Used: Free Dictionary API
API Link: https://dictionaryapi.dev/

Description:
This web app allows users to search for the definition, part of speech, and pronunciation of English words.

Instructions:
1. Open "index.html" to view the group portfolio.
2. Click "Go to Our Dictionary App" to open the app.
3. Type any English word and click "Search" to fetch its meaning.
4. The app displays definitions directly from the Free Dictionary API.

- index.html (Group Portfolio)
